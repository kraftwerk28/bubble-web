# bubble-web
Just cute css style template

## syntax:

**Element** | **Syntax**
---|---
*button* | `<button [class="wave-effect"]><button>`
*text input* | `<div class="input-text" [placeholder=""]></div>`
*range input* | `<input type="range>`
*dropdown menu* | `<div class="dropdown"><ul><li></li>[<li></li>...]</ul></div>`
